# Practica-Asesoria-16-11-2025
La empresa ficticia Gamarra UE desea desarrollar una plataforma digital para la venta de ropa y utensilios producidos por pequeños comerciantes del emporio comercial de Gamarra. La plataforma permitirá a los comerciantes publicar sus productos y a los clientes comprarlos mediante un sistema de carrito de compras y órdenes.

Tú eres parte del equipo encargado de implementar el backend de la plataforma, utilizando Java como lenguaje principal y aplicando buenas prácticas de desarrollo de software.


Para levantar tanto backend, db y frontend ejecutar:

docker-compose up -d
