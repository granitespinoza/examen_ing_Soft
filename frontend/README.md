# Gamarra UE E-commerce Frontend

Frontend desarrollado con Vite, React y TypeScript.

## Tecnologias

- Vite + React + TypeScript
- React Router DOM
- Axios
- Zustand (State management)
- TanStack Query
- Tailwind CSS

## Instalacion

npm install

## Ejecutar

npm run dev

## Estructura

- src/components/ - Componentes reutilizables
- src/pages/ - Paginas
- src/services/ - Servicios API
- src/store/ - Estado global
- src/types/ - Tipos TypeScript
